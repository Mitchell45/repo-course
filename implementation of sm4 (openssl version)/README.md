### 代码说明

对sm4在openssl库中调用和组装生成可以执行的基于openssl库的sm4模块，可以作为后续sm4底层模块加速的对照。

### 运行指导

将源码clone到本地运行main函数即可运行。

软件环境：Visual Studio 2019

硬件环境：PC机
