### 代码说明

对sm4的结构进行了底层的函数实现，经过调用和组装生成可以执行的sm4模块，由于硬件环境有限且没有加速部件，所以运行起来相对比较慢，可以作为后续openssl库函数调用实现和加速的依据。

### 运行指导

将源码clone到本地运行main函数即可运行。

软件环境：Visual Studio 2019

硬件环境：PC机
